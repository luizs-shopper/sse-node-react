## Como usar:
1. Inicie o backend:
  ```bash
    cd server && yarn node server.js
  ```
2. Inicie o frontend:
  ```bash
    cd client/src && yarn start
  ```
3. No frontend digite algum texto no input e clique em "Iniciar"
